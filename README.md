# Math4Ling-Ling (ЛЕТО 2026)
Графы отечественной лингвистики! (Правда, не всей) 
Проект строит и анализирует два графа научного сообщества российских лингвистов на основе данных из открытого репозитория [OpenAlex](https://openalex.org/).

## Что делает проект?

По статьям из трёх ведущих отечественных лингвистических журналов строятся два графа:

- **Граф цитирования** — направленный граф, ребро от A к B означает, что автор A ссылается на работу автора B.
- **Граф соавторства** — ненаправленный граф, ребро между A и B означает, что авторы написали хотя бы одну статью совместно.

Каждый граф разбивается на тематические сообщества двумя алгоритмами:

- **Label Propagation** (граф цитирования)
- **Girvan–Newman** (граф соавторства)

Для каждого сообщества извлекаются ключевые слова через **TF-IDF** + поиск тематически близких кластеров через **KD-Tree**. Итоговые графы экспортируются в формат GEXF и визуализируются в [Gephi Lite](https://gephi.org/gephi-lite/).

-**Хотим посмотреть, пишут ли лингвисты в соавторстве с теми, кого цитируют.**
